# Windows Login Issue Troubleshooting Guide

## Overview
Panduan ini membantu pengguna mengatasi masalah login pada sistem operasi Windows, seperti gagal login, layar freeze, atau masalah akun pengguna.

---

## Gejala
Beberapa indikasi masalah login Windows:

- Muncul pesan **"Password incorrect"**
- Muncul pesan **"We can't sign in to your account"**
- Login loop kembali ke layar login
- Layar menjadi hitam atau freeze setelah login
- Keyboard tidak berfungsi di layar login
- Muncul pesan domain tidak tersedia

---

## Kemungkinan Penyebab
- Password salah
- Keyboard tidak berfungsi
- Profil pengguna rusak
- Driver bermasalah
- Komputer tidak terhubung ke jaringan domain
- Malware atau sistem tidak stabil

---

## Troubleshooting Steps

### 1. Periksa Keyboard
Pastikan keyboard berfungsi dengan baik.

Langkah:
- Coba gunakan keyboard lain
- Gunakan **On-Screen Keyboard** dari ikon **Ease of Access** di layar login

---

### 2. Login Menggunakan Akun Lokal
Jika login domain gagal, coba gunakan akun lokal.

Langkah:
1. Pilih **Other User** pada layar login
2. Masukkan:
    .\username
3. Masukkan password akun lokal

---

### 3. Gunakan Ctrl + Alt + Del
Jika layar login tidak responsif:

- Tekan `Ctrl + Alt + Del`
- Pilih **Restart** atau **Task Manager**

---

---

### 5. Refresh Driver Grafis
Jika layar hitam muncul setelah login:
Tekan: 
    Windows + Ctrl + Shift + B

---

## Hubungi IT Support Jika
- Tidak dapat login dengan akun lokal maupun domain
- Profil pengguna rusak
- Keyboard tidak berfungsi di BIOS atau Windows
- Komputer tidak terhubung ke domain
