# Printer Not Detected in Windows

## Overview
Masalah printer tidak terdeteksi di Windows sering terjadi setelah restart komputer, update Windows, atau perubahan koneksi jaringan. Panduan ini membantu pengguna mengidentifikasi dan memperbaiki masalah tersebut.

---

## Gejala
- Printer tidak muncul di Devices and Printers
- Status printer Offline
- Tidak bisa mencetak dokumen
- Printer tidak muncul di aplikasi

---

## Kemungkinan Penyebab
- Printer tidak menyala
- Kabel USB tidak terhubung
- Printer tidak terhubung ke jaringan
- Driver printer belum terinstall atau rusak
- Print Spooler service berhenti

---

## Troubleshooting Steps

### 1. Periksa Printer
- Pastikan printer menyala
- Periksa kabel USB atau koneksi WiFi
- Restart printer

---

### 2. Restart Print Spooler Service
1. Tekan `Win + R`
2. Ketik `services.msc`
3. Cari **Print Spooler**
4. Klik **Restart**

---

### 3. Periksa Driver Printer
1. Buka **Device Manager**
2. Cari printer
3. Update atau install ulang driver

---

### 4. Tambahkan Ulang Printer
1. Buka **Settings**
2. Pilih **Printers & scanners**
3. Klik **Add device**
4. Pilih printer

---

## Hubungi IT Support Jika
- Printer tetap tidak terdeteksi
- Driver gagal diinstall
- Printer jaringan tidak dapat diakses
